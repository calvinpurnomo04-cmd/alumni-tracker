# Sistem Pelacakan Alumni

## Deskripsi
Sistem web sederhana untuk melacak status alumni berdasarkan data yang dimasukkan oleh admin.

## Fitur
- Input data alumni
- Menampilkan daftar alumni
- Simulasi pelacakan alumni
- Status alumni (Ditemukan, Perlu Verifikasi, Tidak Ditemukan)

## Teknologi
- HTML
- CSS
- JavaScript

## Cara Menjalankan
1. Download project
2. Buka file index.html di browser

| No | Aspek Kualitas | Pengujian | Hasil |

|2|Functionality|Menjalankan pelacakan alumni|Berhasil|
